==============================================================================
 FIFA WORLD CUP 2026 ASSISTANT — DIFFICULT Q&A TEST SET
==============================================================================

PURPOSE
-------
A set of HARD questions and grounded answers for testing the multi-agent
LangGraph workflow. Every question is deliberately "cross-domain": answering it
well needs information that lives behind MORE THAN ONE specialist agent
(Team / History / Prediction / Schedule / General) and their MCP tools.

All answers are grounded ONLY in the project's own data files:
   data/team_knowledge.json   (teams: coach, captain, players, group, titles)
   data/world_cup_history.csv  (past editions: winners, scorers, hosts)
   data/world_cup_matches.csv  (historical match results / stages)
   data/fifa_rankings.csv      (current FIFA ranking points)
   data/world_cup_schedule.csv (2026 fixtures, venues, groups)

NOTE ON "MULTI-AGENT": the supervisor routes each query to one PRIMARY agent,
but these questions are written so a complete answer combines data that several
agents/tools own. Each item lists the agents + MCP tools needed. The prediction
"strength scores" below are computed with the project's real formula:
   score = (0.70*ranking + 0.20*titles + 0.10*recent_pedigree) * 100

 SECTION A — TWO-AGENT QUESTIONS
==============================================================================

------------------------------------------------------------------------------
Q1. "Brazil have won the most World Cups, but are they the favourite to win
     in 2026 based on current form?"
------------------------------------------------------------------------------
Agents: History + Prediction
MCP tools: get_titles_table, get_worldcup_history, get_team_ranking
Difficulty: separates historical pedigree from current-form scoring.

ANSWER:
Yes on both counts. Historically, Brazil are the most successful nation with 5
World Cup titles (1958, 1962, 1970, 1994, 2002) — more than any other country.
On current form they also top the project's predictive model: with 1776 FIFA
ranking points plus maximum historical pedigree, Brazil score about 92.2/100,
the highest of any team. So the data agrees that Brazil are both the most
decorated AND the strongest 2026 contender — though Argentina (~88) and
Germany (~83.6) are close behind. (Analytical estimate, not betting advice.)

Agents: Team + History
